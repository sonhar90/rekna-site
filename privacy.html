Hvem vi er

Rekna er utviklet av Sondre Vaage Haram. Appen hjelper deg med å registrere kvitteringer og generere reiseregninger.

Data vi samler inn

  • Kvitteringsbilder: Når du skanner en kvittering, sendes bildet til Anthropics API (Claude) for automatisk tekstgjenkjenning. Bildet brukes kun til å lese dato, leverandør og beløp, og lagres ikke av Anthropic. Se Anthropics personvernpolicy.
  • Lokal data: Kvitteringer, kjørebokoppføringer og prosjekter lagres utelukkende på din enhet.
  • iCloud-backup (valgfritt): Dersom du aktiverer iCloud-sikkerhetskopi, lagres data i din private iCloud-konto. Ingen andre har tilgang.
  • Kjøpsinformasjon: Abonnement håndteres av Apple via App Store. Vi mottar aldri betalingsinformasjon.

Data vi ikke samler inn

  • Ingen brukerkontoer eller pålogging
  • Ingen annonser eller reklamesporing
  • Ingen analyseverktøy (ingen Google Analytics, Firebase e.l.)
  • Ingen deling av data med tredjeparter utover det som er nevnt over

Dine rettigheter

All data ligger på din enhet. Du kan slette all data ved å avinstallere appen. For henvendelser, kontakt oss på privacy@rekna.app.

Endringer

Vi kan oppdatere denne erklæringen dersom appen endres. Oppdatert versjon vil alltid finnes på denne siden.
